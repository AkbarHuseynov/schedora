# Location Picker - Quick User Guide

## For Shop Owners üè™

### How to Set Your Shop Location

1. **Log In** to your owner account
2. **Go to Shop Setup** (from Owner Dashboard)
3. **Scroll to "Location & Map Settings"** section
4. **Click "Select Location on Map"** button

### Using the Location Picker

When the map modal opens:

**Option A: Click on Map (Easiest)**
- See a map of your shop area
- Click anywhere on the map where your shop is
- A golden marker appears at that spot
- Latitude and Longitude badges turn green
- Click **"Confirm Location"** button
- Done! Coordinates are saved

**Option B: Type Coordinates (Manual)**
- Use the coordinate input fields
- Format: Latitude (40.7128), Longitude (-74.0060)
- Or get them from Google Maps (right-click location)

### Tips
- The map defaults to New York City area
- You can zoom in/out with + and - buttons
- Drag the map to move around
- For more precision, zoom in before clicking
- The marker shows coordinates when you hover over it

### After Setting Location

1. **Check "Show map to clients"** ‚úì
   - This makes your location visible to clients

2. **Check "Show distance"** (optional) ‚úì
   - Clients can see how far away you are
   - Only works if they share their location

3. **Click "Save Changes"**
   - Your location is now saved!

## Location Formats

### Valid Coordinates
```
New York:       40.7128, -74.0060
London:         51.5074, -0.1278
Tokyo:          35.6762, 139.6503
Sydney:        -33.8688, 151.2093
```

### Format Rules
- **Latitude**: -90 to +90
  - North is positive
  - South is negative
- **Longitude**: -180 to +180
  - East is positive
  - West is negative
- Use decimal points (not commas between numbers)

### Getting Coordinates
1. Open Google Maps (maps.google.com)
2. Find your location
3. Right-click on the red marker
4. Click the coordinates that appear at top
5. Copy and paste into form

## Troubleshooting

### Map Not Showing?
- Check your internet connection
- Try refreshing the page
- Make sure JavaScript is enabled in browser

### Can't Find My Location?
- Use the zoom buttons (+ / -)
- Drag the map to navigate
- Or type coordinates manually

### Coordinates Not Saving?
- Click "Confirm Location" button in the modal
- Check that coordinates appear in input fields
- Then click "Save Changes" at bottom

### Location Not Visible to Clients?
- Make sure you checked "Show map to clients"
- Make sure you clicked "Save Changes"
- Wait a moment, then refresh
